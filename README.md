# YC Directory

A startup pitch platform inspired by Y Combinator, built with Next.js 15, Sanity CMS, and NextAuth.

## Features
- GitHub Authentication
- Submit startup pitch ideas
- Browse and search startups
- View startup details
- User profiles
- Real-time updates

## Tech Stack
- Next.js 15
- React
- Tailwind CSS
- Sanity CMS
- NextAuth.js
- Vercel

## Live Demo
https://yc-directory-alpha-gules.vercel.app

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Add your `.env.local` file
4. Run: `npm run dev`
